# customer-churn-sql-analysis
Customer churn analysis using PostgreSQL, SQL queries and business insights.
# Customer Churn Analysis

## Business Problem
The company is losing customers every month. 
The objective is to identify the causes of churn.

## Dataset
- 7000 customers
- subscription data
- usage statistics
- churn information

## Tools
- PostgreSQL
- SQL
- Power BI

## Key Analysis
- Churn rate
- Churn by plan
- Churn by country
- Usage vs churn

## Insights
- Low usage customers are 3x more likely to churn
- Basic plan has the highest churn rate

## Recommendations
- Improve onboarding
- Offer retention discounts
